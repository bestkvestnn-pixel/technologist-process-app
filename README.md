# Технолог

Стартовая страница проекта для приложения разработки технологических процессов по чертежу, эскизу или 3D-модели.

## Что умеет приложение

- Загружает исходник: PDF, JPG/JPEG, PNG, SVG, STEP/STP, IGES, STL, OBJ и другие модели.
- Формирует отчёт по исходнику и заполняет графы детали.
- Подключает агента инженер-технолога для анализа чертежа и разработки ТП.
- Создаёт технологический процесс и комплект документов по ГОСТ 3.1118-82.
- Подбирает инструмент и оптимальные диаметры фрез/сверел по локальным каталогам.
- Формирует черновой G-код для NC 210 и Sinumerik.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 5173
```

Затем откройте:

```text
http://localhost:5173
```

## Рабочий сценарий

1. Нажать **Начать разработку ТП**.
2. Загрузить чертёж, эскиз или модель.
3. При необходимости вставить распознанный текст с чертежа.
4. Нажать **Сформировать отчёт**.
5. Проверить заполненные графы и замечания агента.
6. Нажать **Создать ТП**.
7. При необходимости сформировать G-код.
